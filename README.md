# Warhammer 40k Battle Tracker

A mobile-first Progressive Web App (PWA) for tracking Warhammer 40,000 matches. Support for army list imports, weapon statistics, and real-time game state tracking.

## Getting Started (for another machine)

If you are logging in from another machine and want to continue development, follow these steps:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/jamiet440-beep/warhammer-battle-tracker.git
    cd warhammer-battle-tracker
    ```

2.  **Install Dependencies**:
    Make sure you have Node.js installed, then run:
    ```bash
    npm install
    ```

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

## Project Features

- **Battlescribe Support**: Upload `.rosz` files to import your army lists.
- **Wahapedia Integration**: Local data for unit stats and stratagems.
- **Real-time Tracking**: Manage Command Points (CP) and Victory Points (VP) during the game.

## Tech Stack

- **Framework**: React + Vite
- **Language**: TypeScript
- **State Management**: Zustand
- **Styling**: Vanilla CSS
